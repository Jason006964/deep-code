# 0201学习笔记
## html ## 
* **对于中文网页需要使用 <meta charset="utf-8"> 声明编码，否则会出现乱码（或GBK）**

* html/.htm两种html文档得后缀名没有区别，都可以使用

* ##### <标签>内容</标签> #####
```<html>
&emsp;<head>
&emsp;&emsp;<title>标题（页面）</title>
&emsp;</head>

&emsp;<body>
&emsp;&emsp;<h1>这是一个标题</h1>
&emsp;&emsp;<p>这是一个段落。</p>
&emsp;&emsp;<p>这是另外一个段落。</p>
&emsp;</body>
</html>
```
**运行时显示body部分**


* <!DOCTYPE>声明有助于浏览器中正确显示网页。
